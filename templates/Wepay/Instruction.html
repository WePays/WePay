How to use WePay

  • How to create bill
  • How to pay by cash
  • How to pay by Internet Banking
  • How to fetch chain id via omise

    • How to create bill

    Follow the instructions to create bill:

      1. User can create bill by click + button at bottom right

      2. Add all of the detail of bill

      3. User can more than one topic.

      4. User can create bill by clicking create button at bottom right.

      5. Bill will show up on the bill list and send an email to notify a payers.

    How to pay by cash

      1. Click pay button on the payment page

      2. Select payment type to cash then click pay button.

      3. Status will change to PENDING.

      4. Header needs to verify the bill by confirm or reject.

      5. After confirm the bill.

      6. Status will change to paid and needed to click verify to closed the bill.

      7. It will move to bill history page.

    How to pay by Internet Banking

      1. Header must have the chain id first

      (How to fetch chain id)

      2. After header have chain id payers can now select more payment type.

      3. After select pay, It will bring payer to Omise payment page.

      4. Payment success, the bill that we have paid will disappeared and showed on history page.

    How to fetch chain id via omise -> Get chain ID

    Follow the instructions below to enable account chaining:

      1. Sign up/ Sign in via Omise:


      To Sign up or Sign in via Omise, you must use the same email address that you used to login to WePay.

      2. Visit Omise Dashboard Settings

      3. Click the checkbox enabling account chaining,
      and set the Return url to which the sub-merchant will be redirected after authorization.

      4. Click Update settings and provide the generated Chain url to your sub-merchants so they can authorize account chaining.
      Sub-merchants must visit the Chain Authorization URI, read the terms, and click Authorize.

      5. After that come back then select fetch button, the chain id will show up.
